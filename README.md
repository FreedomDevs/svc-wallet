Вот туториал как создать venv и скачать зависимости:
```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

Или на винде:
```bat
python3 -m venv .venv
.venv\Scripts\activate.bat
pip install -r requirements.txt
```
